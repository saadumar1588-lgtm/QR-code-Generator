# QR Code Generator (Python)

This is a simple Python project that generates a QR code from any text or URL.

## Features
- Takes text or URL as input
- Generates a QR code image
- Saves the QR code as `qrcode.png`

## Requirements
- Python 3
- qrcode library

## How to Run
1. Install the library:
   pip install qrcode[pil]

2. Run the program:
   python scan.py

3. Enter text or URL when asked.

## Output
A QR code image named `qrcode.png` will be generated.
